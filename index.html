🆕 v0.3.3 四连发怎么玩

  1. 先在右侧输入一个任务（比如「写周报」「准备面试」「跑步 5km」），即使没配 API，也会走升级后的本地兜底拆解。
  2. 如果首页有多个进行中的任务，现在点「丢弃」会进入 3 秒二次确认，不再被 popup confirm 卡住。
  3. 进入宠物之家后，喂养会看到 累计喂养 / 今日喂养 / 连续喂养天数 / 亲密度条 / 喂养历史，还有心动动画和彩蛋 toast。
  4. 点顶栏 👤 可查看匿名用户 ID、编辑昵称、导出 / 导入备份，并在支持环境下启用跨设备同步基础。

🍃 本地兜底升级了什么

  1. 输入会先做标准化：去掉「帮我 / 我想 / 谢谢 / 一下」等口头词，统一标点和大小写。
  2. 随后走规则分类：学习 / 写作 / 开会 / 代码 / 生活 / 求职等 20+ 常见意图都能命中。
  3. 如果你反复输入相似任务（如「写周报」「帮我写一下周报」），会尝试复用最近成功 skeleton，让拆解风格更稳定。
  4. 无 API 时仍会产出 4-8 步、带估时的具体动作，而不是一句泛泛的“去完成它”。

🗂 多任务并存怎么玩

  1. 先输入第一个任务并让它开始（比如「写周报」）。
  2. 在分步页点右下角 🗂 切到其他任务 回到首页。
  3. 再输入第二个任务并拆解（比如「打扫桌面」），首页现在会显示 🗂 你的多个任务。
  4. 每张任务卡片显示目标、已完成步数、进度条、创建时间。
  5. 点「继续」切到那个任务；点「丢弃」进入 3 秒二次确认；每个任务独立记录 currentIndex。

👉 提示：本页是 Web 预览，数据存在浏览器 localStorage，不上传任何服务器。
真实 Chrome 扩展下则走 chrome.storage.local，跨会话保持。
