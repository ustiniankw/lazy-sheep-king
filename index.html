🆕 v0.3.4 两大主特性怎么玩

  1. 先在右侧输入一个任务，比如「写周报」「准备评审会」「跑步 5km」，本地会先拆解，再尝试免费 AI 精修。
  2. 点顶栏 👥 创建队伍，复制 6 位团队码给朋友；默认推荐先导出 / 导入队伍快照。
  3. 给当前任务点步骤卡片上的 隐私按钮，可以在公开 / 仅隐藏标题 / 完全隐私之间切换。
  4. 在队友列表里点 🫵 拍一拍，对方下次打开会看到 poke 提醒。
  5. 如果你真想试轻量自动同步，再粘贴 JSONBin.io / npoint.io URL；不填 URL 就完全离线。

📌 Web 试玩说明

右侧是纯前端 Web 预览：没有 chrome.*，因此会自动降级到 localStorage。
同时，Chrome Prompt API 在这个预览环境里通常不可直接使用，所以你会看到“Chrome 内置 AI 不可用”，这是预期行为；真实扩展环境下会按浏览器能力再尝试启用。
