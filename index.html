🍅 番茄钟怎么玩

  1. 右侧输入一件想干的事（比如「写这周周报」），点「拆成傻瓜步骤」。
  2. Plan 页可以微调步骤，点「开始，一步一步做」进入分步页。
  3. 看到 🍅 卡片 → 点「🍅 开始专注」，25 分钟计时开始，红色主题。
  4. 时间到 → 铃声响一下 → 自动切到 5 分钟 ☕ 小憩（绿色主题）。
  5. 小憩结束 → 自动回到专注，累计 🍅 数会自增。
  6. 专注 / 小憩期间可以 ⏸ 暂停 或 ■ 结束。
  7. 时长在 ⚙ 设置页里可以改，专注 1-120 分钟，小憩 1-60 分钟。

🗂 多任务并存怎么玩

  1. 先输入第一个任务并让它开始（比如「写周报」）。
  2. 在分步页点右下角 🗂 切到其他任务 回到首页。
  3. 再输入第二个任务并拆解（比如「打扫桌面」），首页现在会显示 🗂 你的多个任务。
  4. 每张任务卡片显示目标、已完成步数、进度条、创建时间。
  5. 点「继续」切到那个任务；点 ✕ 丢弃一个任务；每个任务独立记录 currentIndex。
