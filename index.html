🆕 v0.3.5 认证账号怎么玩

  1. 点顶栏 👤 我的，顶部就是账号卡片：头像、昵称、模式徽章（👤 访客 / 🔐 本地账号 / 🐙 GitHub）。
  2. 想跨设备找回进度？点 🔐 创建本地密码账号，设昵称 + 密码（至少 6 位、避开弱口令），密码只在本机派生密钥，永不上传。
  3. 本地账号解锁后，导出备份会自动 AES-GCM 加密；导入时输入同一密码即可解密恢复。
  4. 想要真实验证身份？切到 🐙 GitHub 标签，按三步自建免费 OAuth App，把 client_id 粘进来，用设备码在 github.com/login/device 授权。
  5. 生成新用户 ID / 导入备份等敏感操作，在账号锁定时会要求先解锁，保护你的数据。

📌 Web 试玩说明

右侧是纯前端 Web 预览：没有 chrome.*，因此会自动降级到 localStorage。
访客 与 本地密码账号 在试玩环境完全可用；GitHub 登录 因浏览器 CORS 限制可能无法在网页试玩完成，会提示“网页试玩环境无法完成 GitHub 登录，请在安装扩展后使用”，这是预期行为。
