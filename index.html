📅 完成日历打卡图怎么玩

  1. 右侧输入一件想干的事（比如「写这周周报」），拆解后一步一步点 ✅ 完成。
  2. 每完成一步 / 一个任务，都会自动记进当天的打卡日志（stats.dailyLog）。
  3. 点顶栏 📅 进入「完成日历打卡图」。
  4. 顶部 4 张卡片是近 30 天的：完成任务 / 完成步骤 / 连续打卡 / 总养料。
  5. 下方是热力图：每一格代表一天，绿色越深说明当天完成的步骤越多；hover 有当天明细。
  6. 右上角切换 30 天 / 90 天 窗口；连续打卡 = 以今天结尾、连续有步骤完成的天数。

🖼 自定义形象上传（v0.3.2 优化）

  1. 点顶栏 🏠 进入宠物之家，选择 自定义 槽。
  2. 没有图片时，直接点一下就会弹出文件选择器（不再需要长按）。
  3. 也可以用下方的 🖼 上传/更换形象 按钮随时更换。
  4. 点 ✨ 卡通化 一键开关卡通化滤镜。

🗂 多任务并存怎么玩

  1. 先输入第一个任务并让它开始（比如「写周报」）。
  2. 在分步页点右下角 🗂 切到其他任务 回到首页。
  3. 再输入第二个任务并拆解（比如「打扫桌面」），首页现在会显示 🗂 你的多个任务。
  4. 每张任务卡片显示目标、已完成步数、进度条、创建时间。
  5. 点「继续」切到那个任务；点 ✕ 丢弃一个任务；每个任务独立记录 currentIndex。

👉 提示：本页是 Web 预览，数据存在浏览器 localStorage，不上传任何服务器。
真实 Chrome 扩展下则走 chrome.storage.local，跨会话保持。
